# Colocar no GitHub

1. Extraia o ZIP.
2. Crie um repositório vazio no GitHub.
3. Envie todos os arquivos, inclusive as pastas `.github`, `assets`, `collector` e `reports`.
4. Entre em `Settings → Actions → General`.
5. Em **Workflow permissions**, marque **Read and write permissions**.
6. Entre em `Settings → Pages`.
7. Em **Source**, escolha **GitHub Actions**.
8. Abra **Actions → Analisar canal sem API → Run workflow**.
9. Cole o canal e execute.
10. Depois da análise e da publicação, abra o endereço do GitHub Pages.

Não é necessário criar chave, segredo, conta no Google Cloud ou configurar cartão.
