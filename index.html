Descubra os vídeos que puxam o canal para cima.

Este painel não usa a YouTube Data API. Ele abre relatórios gerados pelo coletor local gratuito ou pelo GitHub Actions.

Para analisar outro canal, rode o coletor e abra o arquivo gerado aqui.

python -m collector.collect \
  "https://youtube.com/@canal" \
  --max-videos 200 \
  --output reports/latest.json
